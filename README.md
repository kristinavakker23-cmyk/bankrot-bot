# 🤖 Bankrot Bot

Telegram-бот для ежедневного мониторинга банкротских торгов и поиска высокомаржинальных лотов для перепродажи.

## Возможности

- **Ежедневная рассылка** — каждый день в указанное время
- **Мульти-источник** — парсит ЕФРСБ и агрегаторы
- **Умный анализ** — определяет маржу и приоритет лотов
- **Фильтры** — настройка по цене и категориям
- **Команда /scan** — срочный поиск прямо сейчас

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка Telegram бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте `/newbot`
3. Следуйте инструкциям, получите токен
4. Создайте файл `.env` в корне проекта:

```env
TELEGRAM_TOKEN="ваш_токен_от_BotFather"
CHAT_ID="ваш_chat_id"
```

### 3. Как узнать CHAT_ID

1. Запустите бота и напишите ему `/start`
2. Откройте в браузере: `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
3. Найдите `"chat":{"id":XXXXXX}` — это ваш CHAT_ID

### 4. Запуск

```bash
python bot.py
```

## Настройка фильтров

Отредактируйте файл `config.py`:

```python
# Минимальная и максимальная цена лота
MIN_PRICE = 1000        # ₽
MAX_PRICE = 5000000     # ₽

# Время ежедневной рассылки (МСК)
DAILY_HOUR = 9
DAILY_MINUTE = 0

# Ключевые слова для поиска (ключ -> приоритет, маржа)
CATEGORIES = {
    "ноутбук": {"priority": 1, "margin": 80},
    "сервер": {"priority": 1, "margin": 90},
    # ... добавьте свои
}
```

## Добавление новых источников

Чтобы добавить новую площадку:

1. Создайте файл `src/my_parser.py`
2. Реализуйте функцию `fetch_my_source_lots(session) -> List[Dict]`
3. Добавьте импорт и вызов в `bot.py` в функции `fetch_all_lots()`

## Решение проблем

**Бот не находит лотов:**
- Проверьте подключение к интернету
- Попробуйте расширить фильтры (уменьшите MIN_PRICE)

**Ошибка импорта:**
- Установите зависимости: `pip install -r requirements.txt`

**Telegram API ошибка:**
- Проверьте токен бота
- Убедитесь, что CHAT_ID правильный
- Напишите боту `/start` перед запуском
